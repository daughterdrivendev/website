## Writing Posts

Each post needs frontmatter with:
- `title` - The post title
- `description` - A brief summary for SEO and previews
- `date` - Publication date (YYYY-MM-DD format)
- `draft` - Optional, set to `true` to hide from listing

## Markdown Features

You can use all standard markdown:

- **Bold** and *italic* text
- [Links](https://daughterdriven.dev)
- Code blocks with syntax highlighting

```javascript
const parent = true;
const engineer = true;
const balance = parent && engineer; // challenging but rewarding
```
